# NightCurator
NightCurator is a first person psychological horror game developed in Unity. 
You play as a night security guard in the Marble Museum where strange anomalies occur each shift.
Your goal is identify and neutralize the anomalies before it’s too late. For more info, you can check the Presentation PDF.
